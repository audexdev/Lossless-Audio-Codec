# Support

LAC is an experimental open source project. Support happens through GitHub issues and project documentation.

## Where To Ask

- Bug reports: open a GitHub issue with reproduction steps and environment details.
- Feature requests: open a GitHub issue describing the motivation and compatibility impact.
- Security reports: follow `SECURITY.md` instead of posting exploit details publicly.
- Build and test questions: include the CMake command, compiler, OS, and relevant output.

## Project Scope

The project currently supports a focused PCM WAV domain and an experimental `.lac` format. Unsupported WAV variants, archival compatibility guarantees, and stable public API packaging are still being tracked as future work.
